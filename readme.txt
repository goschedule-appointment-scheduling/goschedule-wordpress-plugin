Embed online appointment scheduling from GoSchedule directly into your site.

== Description ==

GoSchedule is online appointment scheduling software. This plugin allows to embed scheduling directly into their site. Use the [goschedule] short tag on any page
with the options described in the documentation to embed your scheduling site.

== Installation ==

The plugin is simple to install:

1. Download `goschedule.zip`
1. Unzip the file
1. Upload `goschedule` directory to your `/wp-content/plugins` directory
1. Go to the plugin management page and enable the plugin

For more help on installing a Wordpress plugin see http://codex.wordpress.org/Managing_Plugins

== Documentation ==

This creates the [goschedule] tag to embed GoSchedule into the page. Usage:
[goschedule website="https://plugin.goschedule.io/shakthicorp1"]

Options:
website: your scheduling plugin url (required)
width: defaults to 100% (optional)
height: defaults to 600px (optional)
border: defaults to 0 (optional)

For any queries regarding the plugin and customization, kindly drop mail to support@goschedule.io or visit [Help Center](https://help.goschedule.io/)
